What is a Rope?

Imagine you have a massive string, gigabytes in size. Standard strings use contiguous memory arrays. Inserting a character at the beginning takes O(N) time because everything shifts.

A Rope represents a string as a binary tree of smaller strings. Inner nodes contain the weight (length) of the string in their entire left subtree.

Complexity

  • Index Search / CharAt: O(log N)
  • Concatenate / Merge: O(1) or O(log N)
  • Split: O(log N)
  • Insertion / Deletion: O(log N)
